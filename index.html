Dataset: Wild Blueberry Yield Prediction¶

Descrição¶

Conjunto de dados gerado a partir de simulações de polinização de mirtilos utilizando a plataforma GAMMA (https://gama-platform.org/). Os fatores experimentais utilizados são:

  • Tamanho médio dos mirtilos clonados

  • Proporção de abelhas de cada espécie: Honeybee, Bumbles, Andrena e Osmia

  • Informações do tempo: temepratura, precipitação e velocidade do vento

O objetivo do modelo é ser capaz de prever a colheita de uma área polinizada.

Observações¶

  • A variável Clonesize (m²) indica a área em que rede de rizomas (ramos subterraneos) se espalhou. O conjunto de arbustos ligados por esta rede é o chamado clone.

Observamos que há dois grupos correlacionados: Temperaturas e Chuvas. Assim, escolhemos apenas uma váriável de cada grupo para incluir no nosso modelo:

  • Temperaturas -> AverageOfUpperTRange

  • Chuvas -> AverageRainingDays

As váriáveis fruitset, fruitmass e seeds são obtidas junto com a resposta e assim, não seria possível obte-los antes da colheita.

import numpy as np

def mean_absolute_percentage_error(y_true, y_pred):
    y_true, y_pred = np.array(y_true), np.array(y_pred)
    return np.mean(np.abs((y_true - y_pred)/y_true)) * 100

prct_error = mean_absolute_percentage_error(y_true = Y_val, y_pred = preds)
